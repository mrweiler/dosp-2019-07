Before we begin

… we need to generate some data

group_a <- rnorm(n = 30, mean = 5, sd = 5)
group_b <- rnorm(n = 30, mean = 25, sd = 5)
df_wide <- data.frame(group_a, group_b)
df_long <- tidyr::gather(
  df_wide, group_a, group_b,
  key = group, value = score)

Depending on our analysis
will either need data in
long or in wide format

Independent t-test

t-test with data in long format

t.test(formula = score ~ group, data = df_long, var.equal = TRUE)
## 
##  Two Sample t-test
## 
## data:  score by group
## t = -14.09, df = 58, p-value < 2.2e-16
## alternative hypothesis: true difference in means is not equal to 0
## 95 percent confidence interval:
##  -23.51592 -17.66530
## sample estimates:
## mean in group group_a mean in group group_b 
##               4.85866              25.44927

t-test with data in wide format

t.test(x = df_wide$group_a, y = df_wide$group_b, var.equal = TRUE)

Kolmogorov-Smirnov Test

The Kolmogorov-Smirnov Test
uses data in wide format

##  Two-sample Kolmogorov-Smirnov test
## 
## data:  df_wide$group_a and df_wide$group_b
## D = 0.96667, p-value = 1.11e-15

R Notebooks

You need to document your code
or others might not understand
what you were doing

“Others” includes you in
about three weeks from now

Here you can find an R Notebook
of today’s data analysis

R Notebooks let you reuse
the code of an R Script

First you need to label
sections of your R Script

# ---- my first label

You can include your R Script
into your R Notebook or any
other R Markdown document

'''{r}
knitr::read_chunk('analysis/data_analysis.R')
'''

And call sections of your
code using your labels

'''{r my first tag}
'''

More information can be found
in the R Markdown Basics
